Description

This project demonstrates how to blink an LED using an Arduino Uno.

The LED turns on and off at 1-second intervals continuously.

🧰 Components

Arduino Uno

LED

220 ohm resistor

🔌 Connection

Pin 13 → LED long leg (anode)

GND → resistor → LED short leg (cathode)

⚙️ Working Principle

LED turns ON for 1 second

LED turns OFF for 1 second

This cycle repeats forever
