# 🌈 RGB LED Project

##  Description
This project demonstrates how to control an RGB LED using Arduino.

Each color (Red, Green, Blue) is controlled separately, allowing different color combinations.



## 🧰 Components

- Arduino Uno  
- RGB LED  
- 3x 220 ohm resistors  
- Breadboard  
- Jumper wires  



## 🔌 Connection

### Common Cathode RGB LED:

- Long leg → GND  

### Color Pins:

- Red → Pin 9 (with resistor)  
- Green → Pin 10 (with resistor)  
- Blue → Pin 11 (with resistor)  



## ⚙️ Working Principle

- Each color is controlled using digital signals  
- Combining colors creates new colors  
- Example:
  - Red + Green → Yellow  
  - Red + Blue → Purple  
  - All → White
